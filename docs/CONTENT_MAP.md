# CONTENT_MAP

## Джерело контенту
Увесь контент і PLACEHOLDER-и знаходяться в `lib/content.ts`.

## Карта секцій
1. Header
   - Логотип-текст: "Аврора Авто"
   - Навігація: Послуги / Переваги / Процес / Сценарії / Автосервіс / Підхід / FAQ / Контакти
   - CTA: "Отримати розрахунок" (модалка), "Зателефонувати" (tel)

2. Hero
   - H1: "Вантажна логістика та сервіс автотранспорту в ритмі міста"
   - Підзаголовок: опис логістики та сервісу
   - CTA: primary + secondary
   - Пояснювальний note

3. Services
   - 5 карток послуг

4. Value Props
   - 5 тез без цифр

5. Process
   - 4 кроки + SVG маршрут

6. Use Cases
   - Теги сценаріїв та типів вантажів

7. AutoService
   - Окремий блок про сервіс

8. Cases/Approach
   - Нейтральні сценарії без конкретики

9. FAQ
   - 4 питання

10. Contact
   - Телефон, адреса, кнопка "Відкрити в картах"
   - Форма (React Hook Form + Zod)
   - Мапа (lazy iframe)

11. Footer
   - Юридичність, копірайт

12. Legal pages
   - Політика обробки персональних даних
   - Умови зв’язку

## PLACEHOLDER-и
- `PLACEHOLDER_URL`
- `PLACEHOLDER_EMAIL`
- `PLACEHOLDER_PRICE`

Заміна здійснюється у `lib/content.ts`.
