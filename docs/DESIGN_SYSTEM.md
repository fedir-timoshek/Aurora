# DESIGN_SYSTEM

## Візуальна концепція
Night city / neon aurora. Темний фон, холодні неонові акценти, легкі градієнтні світила.
Фонові ілюстрації виконані локальними SVG (aurora veil, cityline, mesh).

## Палітра
- Night 950: #0b0b14
- Night 900: #0f1120
- Night 800: #141833
- Neon 500: #22c5ff
- Aurora 400: #63f0a0
- Magma 400: #ff5bb0

## Типографіка
- Шрифт: Manrope (400, 600, 700)
- Ієрархія: H1 40-60px, H2 28-36px, body 14-18px
- Трекінг: uppercase labels із wide tracking

## Компоненти
- Buttons: pill shape, glow border, hover accent
- Cards: rounded-2xl, border 1px, subtle gradient overlay
- Tags: pill, нейтральний контраст
- Form fields: rounded-xl, border focus neon
- Modal: center, dark overlay, close button
- Illustration badges: локальні SVG з неоновим контуром

## Токени (Tailwind)
- shadow-glow, shadow-glowStrong
- bg-aurora-grid
- animation: aurora-shift
