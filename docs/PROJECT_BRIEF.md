# PROJECT_BRIEF

## Цілі
- Створити 1-page landing для ТОВ «Аврора Авто» з фокусом на вантажні автоперевезення та супутні послуги.
- Підкреслити надійність, системність та технологічний підхід без вигаданих фактів.
- Забезпечити готовність до запуску локально та до деплою без зовнішніх ключів.

## Аудиторія
- Бізнеси, яким потрібні регулярні або разові вантажні перевезення.
- Приватні клієнти для переїздів та перевезення речей.
- Компанії, які потребують технічної підтримки транспорту.

## Ключові меседжі
- Чітка логістика та передбачуваний процес.
- Акуратне ставлення до вантажу і відповідальність у деталях.
- Гнучкі формати співпраці, прозора комунікація.

## Тон і стиль
- Преміальний, сучасний, технологічний.
- Впевненість без гучних обіцянок.
- Лаконічні, зрозумілі формулювання.

## Обмеження
- Жодних вигаданих цифр, дат, клієнтів, відгуків, гарантій.
- PLACEHOLDER-и для точних даних.
- Усі користувацькі тексти українською.
